氨基酸

一、氨基酸的化学结构

基本结构通式

COOH ─── 羧基（酸性）
│
H₂N—C—H ─── α-碳（手性碳，L型）
│
R ─── 侧链（决定氨基酸特性）

特点：

  • α-碳为手性碳（除甘氨酸R=H外）
  • 天然蛋白质氨基酸均为L-型
  • 同时含氨基（碱性）和羧基（酸性）→ 两性电解质

三、氨基酸的等电点（pI）

概念

等电点（pI）：氨基酸分子净电荷为零时的pH值。此时氨基酸在电场中不移动。

计算

  • 酸性氨基酸（Asp, Glu）：pI = (pK₁ + pKR) / 2
  • 碱性氨基酸（Lys, Arg, His）：pI = (pKR + pK₂) / 2
  • 中性氨基酸：pI = (pK₁ + pK₂) / 2 ≈ 5.0~6.5

★ 等电点重点：
- pI < 7：酸性氨基酸（Asp, Glu）
- pI > 7：碱性氨基酸（Lys, Arg, His）
- pI ≈ 6：中性氨基酸
- 在pI时，氨基酸溶解度最小，容易沉淀（可用于蛋白质分离）

四、氨基酸的常见性质

1. 两性解离

氨基酸在溶液中的存在形式取决于pH：

     pH < pI              pH = pI               pH > pI
   H₃N⁺—CH—COOH      H₃N⁺—CH—COO⁻         H₂N—CH—COO⁻
         │                   │                    │
         R                   R                    R
    (阳离子)              (两性离子)            (阴离子)

2. 紫外吸收

  • Trp（280nm，最强）> Tyr（280nm）> Phe（260nm，弱）
  • **应用**：OD₂₈₀ 测定蛋白质浓度（Beer-Lambert定律）

3. 茚三酮反应

  • 氨基酸 + 茚三酮 → 蓝紫色化合物
  • 应用：氨基酸定量检测（570nm比色）
  • 脯氨酸与茚三酮反应显黄色

4. Sanger试剂（DNFB）

  • 与α-NH₂反应 → 黄色DNP-氨基酸
  • 应用：测定蛋白质N端第一个氨基酸

5. Edman降解

  • 异硫氰酸苯酯（PITC）与N端氨基反应
  • 酸解离后释放标记的氨基酸衍生物（PTH-氨基酸）
  • 应用：蛋白质N端测序，自动测序仪原理
