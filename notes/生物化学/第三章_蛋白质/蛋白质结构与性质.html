蛋白质结构与性质

一、蛋白质的结构层次

蛋白质结构的四个层次： 
一级结构         氨基酸序列（肽键连接）
   │                  ↓   （由基因决定，遗传信息直接产物）
   │
二级结构         局部肽段的空间构象（氢键维持）
   │         ┌─ α-螺旋（链内氢键）
   │         ├─ β-折叠（链间/链内氢键）
   │         ├─ β-转角（180°回折）
   │         └─ 无规卷曲
   │
三级结构         整条多肽链的三维空间排布
   │         （疏水作用、氢键、离子键、范德华力、二硫键）
   │
四级结构         多亚基聚合（非共价相互作用）
            （如血红蛋白：α₂β₂四聚体）

二、二级结构详解

结构形状氢键模式特点
α-螺旋右手螺旋n → n+4（C=O···H-N）每圈3.6个残基，螺距0.54nm
β-折叠锯齿状相邻肽段间平行/反平行，反平行更稳定
β-转角180°回折n → n+3常含Pro/Gly
Ω环环状无规律常在蛋白质表面
★ 破坏α-螺旋的氨基酸：Pro（刚性的环，无H给体）、Gly（柔性过大，易弯曲）
★ 有利于α-螺旋的氨基酸：Ala、Leu、Glu
★ 有利于β-折叠的氨基酸：Val、Ile、Tyr

三、三级结构

维持三级结构的驱动力

作用力参与基团强度（kJ/mol）
疏水相互作用非极性侧链（Leu, Val, Phe, Ile）熵驱动（最强）
氢键极性侧链、主链C=O和N-H5-30
离子键（盐桥）Lys⁺-Asp⁻, Arg⁺-Glu⁻20-80
范德华力所有紧密接触的原子0.5-5
二硫键（共价）Cys-S-S-Cys~210

结构域（Domain）

  • 三级结构中独立折叠的功能单位
  • 每个结构域含约100-200个氨基酸
  • 不同结构域执行不同功能
  • 如：DNA结合蛋白的螺旋-转角-螺旋结构域

四、四级结构

特点

  • 由两条或多条多肽链（亚基）通过非共价相互作用聚合
  • 多条相同链：同源多聚体（如肌动蛋白）
  • 多条不同链：异源多聚体（如血红蛋白α₂β₂）

别构效应（Allostery）

  • 一个亚基的构象变化影响其他亚基
  • 导致结合曲线呈S形（协同效应）
  • 如：血红蛋白的O₂结合（正协同）

五、蛋白质的性质

1. 两性解离

  • 蛋白质表面有大量可电离基团（侧链+末端）
  • 等电点(pI)：净电荷为零时的pH
  • pI处蛋白质溶解度最小

2. 变性（Denaturation）

变性因素作用机制
高温破坏弱键（氢键、范德华力）
强酸/强碱破坏离子键，改变基团电荷
尿素/盐酸胍破坏氢键（尤其二级结构）
SDS（十二烷基硫酸钠）疏水作用破坏，带负电使蛋白伸展
β-巯基乙醇还原二硫键
有机溶剂破坏疏水核心

3. 复性（Renaturation）

  • Anfinsen实验（1972诺贝尔奖）：核糖核酸酶A可自发复性
  • 证明：一级结构决定高级结构
  • 但部分蛋白质需分子伴侣辅助折叠

4. 紫外吸收（280nm）

  • 主要由Trp、Tyr贡献
  • 用于蛋白质定量

5. 盐析

  • 高浓度中性盐（(NH₄)₂SO₄、NaCl）使蛋白质沉淀
  • 不破坏蛋白质天然构象（可逆）
  • 不同蛋白质在不同盐浓度沉淀

六、蛋白质的分类

按形状

  • 纤维状蛋白：长条形，结构功能（胶原蛋白、角蛋白）
  • 球状蛋白：球形，功能多样（酶、血红蛋白）

按组成

  • 简单蛋白：仅含氨基酸
  • 结合蛋白：含非蛋白辅基（血红蛋白→血红素、糖蛋白→糖链）

七、蛋白质分离纯化方法

方法原理用途
盐析不同蛋白在不同盐浓度沉淀粗分离
凝胶过滤按大小分离（大者先出）分子量测定、纯化
离子交换按电荷分离纯化
SDS-PAGE按分子量分离分析纯度、MW
亲和层析特异性结合高效纯化
等电聚焦按pI分离分析pI
