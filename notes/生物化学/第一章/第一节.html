第一节：蛋白质的结构与功能

蛋白质是由氨基酸通过肽键连接而成的生物大分子。

一、氨基酸

氨基酸是蛋白质的基本组成单位。标准氨基酸有20种。

二、蛋白质的结构层次

  • 一级结构：氨基酸序列
  • 二级结构：α-螺旋、β-折叠
  • 三级结构：多肽链的空间构象
  • 四级结构：亚基聚合

三、公式示例

水的离子积：Kw = [H⁺][OH⁻] = 1.0 × 10⁻¹⁴

四、代码示例

def calculate_mw(sequence):
    """计算蛋白质分子量"""
    mw_table = {
        'A': 89.09, 'R': 174.20, 'N': 132.12,
        'D': 133.10, 'C': 121.15, 'Q': 146.15,
        'E': 147.13, 'G': 75.07,  'H': 155.16,
        'I': 131.17, 'L': 131.17, 'K': 146.19,
        'M': 149.21, 'F': 165.19, 'P': 115.13,
        'S': 105.09, 'T': 119.12, 'W': 204.23,
        'Y': 181.19, 'V': 117.15
    }
    return sum(mw_table.get(aa.upper(), 0) for aa in sequence)
