蛋白质代谢

一、概述

蛋白质代谢包括：

  • 蛋白质合成（翻译）：氨基酸→多肽链
  • 蛋白质降解：多肽链→氨基酸
  • 氨基酸代谢：脱氨、碳骨架氧化、一碳单位代谢

二、蛋白质合成（翻译）

中心法则

DNA ──[转录]──→ RNA ──[翻译]──→ 蛋白质 ↑tRNA, rRNA

遗传密码

特征说明
三联体密码3个核苷酸编码1个氨基酸
简并性多数氨基酸有多个密码子（如Leu有6个）
起始密码AUG（Met）
终止密码UAA, UAG, UGA（无义密码子）
通用性从病毒到人类基本一致（线粒体略有差异）
摆动假说tRNA反密码子第1位与mRNA第3位可非标准配对

三、翻译过程

1. 氨基酸活化

氨基酸 + tRNA + ATP → 氨酰-tRNA + AMP + PPi

氨酰-tRNA合成酶：高度专一，校对功能（"第二遗传密码"）

2. 翻译起始（原核）

  • 30S小亚基 + IF-1, IF-3 → 结合mRNA的Shine-Dalgarno序列
  • fMet-tRNA + IF-2 + GTP → 识别起始密码AUG
  • 50S大亚基结合 → 起始复合物形成

3. 延伸

延伸的3个循环步骤（EF-Tu, EF-G, GTP）：

  密码子识别 ← 氨酰-tRNA进入A位（EF-Tu+GTP）
      ↓
  肽键形成 ← 肽酰转移酶（23S rRNA核酶活性）
      ↓
  转位 ← 核糖体沿mRNA移动3nt（EF-G+GTP）
      ↑
  重复直至遇到终止密码

4. 终止

  • 释放因子RF1/RF2识别终止密码
  • 多肽链从tRNA释放
  • 核糖体解离为大小亚基

四、蛋白质降解

泛素-蛋白酶体途径（真核细胞质）

泛素化级联反应：

  泛素( Ub ) + E1 ─(ATP)─→ Ub~E1（活化）
                                          ↓
                              E2 ←─ Ub~E2（结合）
                                          ↓
                              E3 ←─ Ub~E3-底物（连接）
                                          ↓
                          多聚泛素链标记底物
                                          ↓
                         26S蛋白酶体识别并降解
                         （产生短肽→进一步水解为氨基酸）

★ 2004年诺贝尔化学奖：Ciechanover, Hershko, Rose

溶酶体途径

  • 通过自噬（Autophagy）吞入胞质成分
  • 溶酶体水解酶（组织蛋白酶）降解
  • 非选择性的，饥饿时加强

五、氨基酸的分解代谢

脱氨基作用

类型反应酶
氧化脱氨基氨基酸 → 酮酸 + NH₃谷氨酸脱氢酶(GDH)
转氨基氨基酸 + α-酮酸 ⇌ 酮酸 + 新氨基酸转氨酶（ALT, AST）
联合脱氨基转氨+氧化脱氨耦合转氨酶 + GDH
非氧化脱氨基脱水/脱硫化氢等Ser/Thr脱水酶
联合脱氨基（主要途径）： 
     氨基酸       α-酮戊二酸      NAD⁺(NADP⁺)
      │              ↑              ↑
      │转氨酶        │              │谷氨酸脱氢酶
      ↓              │              ↓
    α-酮酸       谷氨酸        NH₃ + NADH(NADPH)

★ 转氨酶需要辅酶PLP（磷酸吡哆醛=维生素B₆）
★ ALT（GPT）：丙氨酸+α-KG ⇌ 丙酮酸+谷氨酸
★ AST（GOT）：天冬氨酸+α-KG ⇌ 草酰乙酸+谷氨酸

六、氨基酸碳骨架的氧化

生糖与生酮氨基酸

类别氨基酸
生糖氨基酸Ala, Arg, Asn, Asp, Cys, Glu, Gln, Gly, His, Met, Pro, Ser, Thr, Val
生酮氨基酸Leu, Lys
生糖兼生酮Ile, Phe, Trp, Tyr

碳骨架进入TCA循环的7个入口

    Ala, Cys, Gly, Ser, Thr, Trp ──→ 丙酮酸 ──→ 乙酰CoA
    Asn, Asp ──→ 草酰乙酸
    Ile, Leu, Trp ──→ 乙酰CoA / 乙酰乙酰CoA
    Phe, Tyr ──→ 延胡索酸 + 乙酰乙酰CoA
    Arg, His, Glu, Gln, Pro ──→ α-酮戊二酸
    Ile, Met, Val ──→ 琥珀酰CoA
    Leu, Lys ──→ 乙酰CoA / 乙酰乙酰CoA（纯生酮）

七、一碳单位代谢

定义

一碳单位：含一个碳原子的基团（甲基、亚甲基、甲酰基等），由四氢叶酸(THF)携带。

来源与去向

来源一碳单位去向
Ser, Gly亚甲基(-CH₂-)胸苷酸(dTMP)合成
His甲酰基(-CHO)嘌呤合成（C2, C8）
Gly次甲基(-CH=)嘌呤合成
Met（S-腺苷甲硫氨酸）甲基(-CH₃)甲基化反应
★ 临床意义：甲氨蝶呤(MTX)抑制二氢叶酸还原酶，阻断THF再生，抑制DNA合成→抗癌
