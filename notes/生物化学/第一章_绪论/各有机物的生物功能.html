三、有机物功能的物质基础与结构关系

结构决定功能 — 逻辑框架： 
        分子结构              ──────→         生物功能
      （化学组成与构型）                     （生理作用）

糖类：多羟基醛/酮                   能源 + 结构 + 识别
      ↑                                    ↑
      羟基多 → 好溶剂                  多糖分支多 → 储能效率高
      单糖种类多                        糖链序列 → 细胞识别

脂质：长碳链 + 酯键                  膜 + 储能 + 信号
      ↑                                    ↑
      疏水碳链 → 天然屏障             双层膜 → 选择通透性
      脂肪酸饱和度 → 流动性           固醇 → 膜的流动性调节

蛋白质：20种氨基酸序列                酶 + 结构 + 运输 + 信号
      ↑                                    ↑
      氨基酸序列 → 三维结构         特定空间构象 → 底物专一性
      活性中心 → 催化功能           别构中心 → 活性调控

核酸：碱基序列 + 双螺旋              信息存储 + 表达传递
      ↑                                    ↑
      碱基互补配对 → 精确复制      三联体密码 → 蛋白质合成
      双螺旋稳定结构 → 长期保存
