第一章：门电路

基本逻辑门

门类型逻辑表达式真值表（A B → Y）
与门 (AND)Y = A · B1 1 → 1
或门 (OR)Y = A + B0 1 → 1
非门 (NOT)Y = A'1 → 0
与非门 (NAND)Y = (A · B)'1 1 → 0
或非门 (NOR)Y = (A + B)'0 0 → 1

CMOS 门电路

CMOS 电路由 PMOS 和 NMOS 组成互补对称结构。

  • 静态功耗低：仅在开关瞬态消耗功率
  • 抗干扰能力强
  • 集成度高
module and_gate (
    input  wire A,
    input  wire B,
    output wire Y
);
    assign Y = A & B;
endmodule
